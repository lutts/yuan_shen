## 夜芙万钟（满命夜兰，满命芙芙）

* 手法1：夜兰e 芙芙qeaa 万叶e 芙芙zaaz，万叶e，夜兰aaa直到芙芙q准备好
* 手法2：芙芙q，万叶q，芙芙eaa 夜兰eqe，芙芙zaaz, 万叶e, 夜兰aaaeaa直到芙芙q准备好

手法1的优点是夜兰的e好得更快，因此充能循环更快，
